ELK部署

发表于 2018-09-01 | 分类于 大数据

1、ElasticSearch服务器的搭建

1.1在127.0.0.1主机中下载ElasticSearch

下载压缩包

wget https://artifacts.elastic.co/downloads/elasticsearch/elasticsearch-6.2.2.tar.gz

解压（elasticsearch不能在root路径下，应在/app,/opt等非root下，否则运行报错）

tar -xzf elasticsearch-6.2.2.tar.gz

1.2 配置用户组：

创建elsearch用户组及elsearch用户

groupadd elsearch

useradd elsearch -g elsearch -p elasticsearch

更改elasticsearch文件夹及内部文件的所属用户及组为elsearch:elsearch

chown -R elsearch:elsearch elasticsearch-6.2.2

1.3打开端口号(127.0.0.1)

firewall-cmd –zone=public –add-port=9200/tcp –elasticsearch

firewall-cmd –reload

或

vi /etc/sysconfig/iptables

-A INPUT -m state –state NEW -m tcp -p tcp –dport 9200 -j ACCEPT

service iptables restart

1.4配置文件：

vi /etc/sysctl.conf

增加一行 vm.max_map_count=655360

接着执行 sysctl -p

vi /etc/security/limits.conf

添加

* - nofile 65536

* - memlock unlimited

wq保存

vi /etc/security/limits.d/90-nproc.conf

将内容改为：

* soft nproc 4096

如果是使用xshell开两个窗口的话修改完成之后一定要断开重新登录一下

切换到elsearch用户再启动

cd elasticsearch-6.2.2

su elsearch

bin/elasticsearch

curl -H “Content-Type: application/json”-XPUT http://127.0.0.1:9200/9airblacklist/_settings-d ‘{ “index” : { “max_result_window” : 2000000000}}’

2.安装logstash

2.1在127.0.0.1主机中下载Logstash

wget https://artifacts.elastic.co/downloads/logstash/logstash-6.2.2.tar.gz

tar -xzf logstash-6.2.2.tar.gz

cd logstash-6.2.2

启动判断是否运行成功

bin/logstash -e ‘input { stdin { } } output { stdout {} }’

运行成功则退出

2.2安装Logstash-input-jdbc插件并配置同步数据

bin/logstash-plugin install logstash-input-jdbc

cd bin

mkdir config-mysql

cd config-mysql

vim mysql.conf

input {

}

output {

}

保存后运行同步程序

nohup ./logstash -f config-mysql/mysql.conf &

3.安装Kibana

3.1 kibana下载(127.0.0.1)

wget https://artifacts.elastic.co/downloads/kibana/kibana-6.2.2-linux-x86_64.tar.gz

tar -xzf kibana-6.2.2-linux-x86_64.tar.gz

cd kibana-6.2.2-linux-x86_64

3.2 修改配置

vi config/kibana.yml

将默认配置改成如下：

server.port: 5601

server.host: “0.0.0.0”

elasticsearch.url: “http://localhost:9200“

kibana.index: “.kibana”

保存

3.3 运行

bin/kibana

浏览器打开http://127.0.0.1:5601访问

安装成功
